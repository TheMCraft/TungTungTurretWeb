Projektinformationen

Das "Tung Tung Turret" Projekt ist eine Schularbeit im Rahmen des Unterrichts an der HTL Klagenfurt. Es handelt sich um ein Bildungsprojekt zur Demonstration von Embedded Systems, Robotik und Web-Technologien.

Server-Log-Dateien

Der Provider der Seiten erhebt und speichert automatisch Informationen in so genannten Server-Log-Dateien, die Ihr Browser automatisch an uns übermittelt. Dies sind:

Technologie-Stack

Dieses Projekt verwendet folgende Technologien:

  • ESP32-CAM - Mikrocontroller mit integrierter Kamera
  • HTML5, CSS3, JavaScript - Frontend-Technologien
  • Bootstrap 5 - CSS Framework
  • WebSocket - Echtzeitkommunikation
  • Arduino IDE - Entwicklungsumgebung
  • 3D-Druck - Mechanische Komponenten
